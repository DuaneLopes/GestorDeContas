# GestorDeSenhas

É um sofware criando com o objectivo de ajudar a manter seguro e organizado os teus dados de login.
Possui uma interface simples e intuitiva.
Os dados são encriptados usando O AES de 16 bits.

#Funcionalidades
 * Adicionar uma conta de usuario
 * Adicionar os dados de login das suas contas
 * Editar e actualizar os dados das contas
 * Eliminar a conta

#Dependencia
-------------
sqlite-jdbc.jar (Faça o download da versão mais recente)
